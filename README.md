A fast-paced, addictive 2D browser arcade game built from scratch using HTML5, CSS3, and modern Vanilla JavaScript.

## 🎮 How to Play & Rules
The goal is simple: Survive the falling ball storm and score as high as possible!

- 🗑️ **Control the Basket:** Use the **Left and Right Arrow Keys** on your keyboard to move the basket.
- 🟢 **Catch Balls:** Catch the falling balls in the basket to increase your score.
- 💥 **Avoid Spikes:** The floor is covered with sharp spikes. If a ball hits the spikes, it explodes instantly and you **lose a life**.
- 🌟 **Combo Bonus:** Successfully catch **7 balls in a row** without letting any hit the spikes to earn a **Bonus Life**!

---

## ✨ Features & JS Logic Implemented
- 🕹️ **Dynamic Game Loop:** Handles smooth continuous object dropping and collision detection.
- 🔢 **State Management:** Tracks lives, scores, and combo streaks in real-time.
- ⌨️ **Event Listeners:** Responsive keyboard controls for fluid basket movement.
- 🎨 **Interactive UI:** Dynamic life counter display at the top of the screen.

---

## 🛠️ Technologies Used
- **HTML5 Canvas / DOM Elements** - For game rendering.
- **CSS3** - For the atmospheric background and layout styling.
- **JavaScript (ES6+)** - For the complete game logic, physics, and mechanics.
